Intermediate Concepts in Python

Intermediate Concepts in Python build on the basics to help you write more powerful and efficient programs. These include understanding and using object-oriented programming (OOP) principles like classes and inheritance, working with modules and packages to organize code, and handling exceptions for better error control.

Intermediate

Classes and Objects

Objects are an encapsulation of variables and functions into a single entity.

Objects get their variables and functions from classes.

Classes are essentially a template to create your objects.

Assign the above class(template) to an object :

script.py

class MyClass:
    variable = "blah"

    def function(self):
        print("This is a message inside the class.")

myobjectx = MyClass()

Accessing Object Variables

To access the variable inside of the newly created object "myobjectx":

script.py

class MyClass:
    variable = "blah"

    def function(self):
        print("This is a message inside the class.")

myobjectx = MyClass()

print(myobjectx.variable)

Python Shell (Result)

blah

You can create multiple different objects that are of the same class(have the same variables and functions defined). However, each object contains independent copies of the variables defined in the class.

script.py

class MyClass:
    variable = "blah"

    def function(self):
        print("This is a message inside the class.")

myobjectx = MyClass()
myobjecty = MyClass()

myobjecty.variable = "yackity"

# Then print out both values
print(myobjectx.variable)
print(myobjecty.variable)

Python Shell (Result)

blah

yackity
