Installing Python and Setting Up an IDE

macOS

Step-by-Step Guide to Install Python on macOS

  1. Download the Installer

    Visit the official Python downloads page and download the latest version suitable for macOS. The installer is a .pkg file, which is a standard macOS installer package.

  2. Run the Installer

    Locate the downloaded .pkg file and double-click it to launch the installer. Follow the on-screen instructions to complete the installation.

  3. Verify the Installation

    After installation, open the Terminal application and run:

    python3 --version

    This command should display the installed Python version, confirming a successful installation.

  4. Access Python's Interactive Shell (IDLE)

    You can start Python's integrated development environment (IDLE) by typing idle3 in the Terminal or by locating IDLE in your Applications folder.

Windows

Step-by-Step Guide to Install Python on Windows

  1. Download the Installer

    Visit the official Python downloads page and download the Windows installer (64-bit) from the "Stable Releases" section.

  2. Run the Installer

    Locate the downloaded .exe file and double-click it to launch the installer. Follow the on-screen instructions to complete the installation.

  3. Verify the Installation

    After installation, open the Command Prompt and run:

    python --version

    This command should display the installed Python version, confirming a successful installation.

  4. Access Python's Interactive Shell (IDLE)

    You can start Python's integrated development environment (IDLE) by typing python in the Command Prompt. This will launch the interactive shell, and you'll see something like:

    >>>

    You can now type Python code directly.

Variables and Types

Python is completely object oriented, and not "statically typed". You do not need to declare variables before using them, or declare their type.

Numbers

Python supports two types of numbers - integers(whole numbers) and floating point numbers(decimals).

script.py

myint = 7

print(myint)

Python Shell (Result)

7

script.py

# Two notations to define a floating point number

myfloat = 7.0

print(myfloat)

myfloat = float(7)

print(myfloat)

Python Shell (Result)

7.0

7.0

Strings

Strings are defined either with a single quote or a double quotes.

script.py

mystring = 'hello'

print(mystring)

mystring = "hello"

print(mystring)

Python Shell (Result)

hello

hello

Using double quotes makes it easy to include apostrophes (whereas these would terminate the string if using single quotes)

script.py

mystring = "Don't worry about apostrophes"

print(mystring)

Python Shell (Result)

Don't worry about apostrophes
